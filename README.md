HC12NET
byGozzi
HC-12 wireless serial port communication module is a new-generation multichannel embedded wireless data transmission module. Communication distance is 1000m in open space. The baud rate can be set to be 1,200bps, 2,400bps, 4,800bps, 9,600bps, 19,200bps, 38,400bps, 57,600bps, and 115,200bps. This library manages the connection of one HC12 (Master) connected to multiple HC12 (Slaves). Master send request to all slaves and only one slave answer. Master can request an int value, float value or String value. Master can send command to slave too.
